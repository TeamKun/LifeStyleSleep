# LifeStyleSleep
- 時刻に応じて寝る 
- 寝ている間は動けない  
- 寝る20分前(現実時間1分)のときのみベッドを使用できる  
- 寝る時間にベッドにいないと立ったまま寝る  
- sleep・awakeオブジェクティブに寝る・起きる時間をそれぞれ登録　　
- デフォルトは22時~5時  
- ジョイン・リスポーン・サーバーリロードの時点のみ睡眠時間内であっても無視される  
- 時間を加速させたい場合は以下のコマンドを使用  
    - /<speed-up> [stop, default, high, ultra]
    - それぞれ0倍, 1倍, 2倍, 10倍
